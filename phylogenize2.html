phylogenize2

A robust phylogenetic regression framework for microbiome analysis

Nghi (Nia) Tran, Kathyrn Kananen, Patrick Bradley

Project Overview

phylogenize is a R-based and conda package to utilize PGLS (phylogenetic generalized least square) to use gene contents to predict for environmental phenotypes (such as specificity, prevalence or abundance) directly from sequencing data, while also accounting for the evolutionary relationships between microbes.

Methods

We use Phylogenetic Generalized Least Square (phylolm package), where predictor variables are gene presence/absence dataset (generated from UniRef, UHGP, and GTDB dataset clustering with Anivo's mmseq). and outcome variables are count data (processed with QC, BBDuk, Kraken/Bracken), which will be transformed into phenotype data (prevalence/specificity or abundance, depending on user's choice)

To account for outlier and non-parametric datasets, we implemented phylogenetic independent contrast - PIC (R ape package), combined with regression with empirical null distribution estimation method ("permulate"). This will help to reduce FPR and increase sensitivity. We use early-stopping (an adaptive machine-learning techniques to halt model when no further signficant hits are found) that remarkably save computing runtime and resources.
